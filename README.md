# College Connect Web Client

The modern web frontend for College Connect, built to deliver a premium user experience for students, alumni, and administrators.

## ‚ú® Key Features

*   **Responsive Dashboard**: A centralized hub for latest updates, stats, and quick actions.
*   **Smart Search Interface**:
    *   Toggle between Standard and **AI Semantic Search**.
    *   Filter by role, department, skills, and more.
*   **Profile Management**:
    *   Rich profile editing with **Resume Upload** functionality.
    *   Automatic data extraction from uploaded resumes.
*   **Support Portal**:
    *   Dedicated support page with ticket tracking.
    *   Global **Floating Action Button (FAB)** for quick help access.
*   **Admin Tools**:
    *   Exclusive dashboards for College Admins and Super Admins to manage the platform.
*   **Design**:
    *   Built with **Material UI (MUI)** for a sleek, consistent, and accessible design system.
